## Chatbot demos implemented by different models

**chat_Mistral7B.py** — Mistral 7B  
**chat_Qwen0.5B.py** — Qwen2.5Coder1_5B_Instruct or Qwen2.5_0.5B_Instruct_GPTQ_Int4  
**jsonl2json.py** — Convert jsonl chat history into recent json that can be loaded by chat  
**wandbTest.py** — Test connection of wandb  
**qlora_qwen_wandb.py** — PEFT LoRA test  
**qlora_inference.py** — PEFT LoRA inference test, output results to wandb table  
**VecDB_bgem3.py** — Build vector database with bgem3 embedding and save to local  
**VecDB_miniLM.py** — Build vector database with miniLM embedding and save to local  
**QA_RAGminiLM.py** — RAG-based QA system with miniLM-based vector database  
**QA_bgem3.py** — RAG-based QA system with bgem3-based vector database  
**Chat_RAG.py** — RAG-based QA without saving vecDB  
**Chat_RAG_vecDB.py** — RAG-based QA and VecDB generate/save section  
**Chat_RAG_vecDB_mulSession.py** — RAG-based QA with multiple session and VecDB generate/save section
